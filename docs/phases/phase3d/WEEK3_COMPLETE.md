## Confidence Scoring (Unified)

**Formula:**
```
Base: 99.5% (Deterministic Analysis)
  × (1 + Architect adjustment)    (-0% to -10%)
  × (1 + Tester adjustment)       (-0% to -5%)
  × (1 + Red Team adjustment)     (-0% to -10%)
= Final: 89-99.5%
```

**Example:**
```
99.5% base
  × 0.98 (Architect: -2%, minor design gaps)
  × 0.99 (Tester: -1%, MITRE mappings valid)
  × 0.97 (Red Team: -3%, moderate exploit difficulty)
= 93.6% final confidence
```

**Interpretation:**
- **95-100%:** Exceptional — Minimal gaps
- **90-95%:** Excellent — Minor gaps, solid recommendations
- **85-90%:** Good — Some gaps, recommendations valid
- **80-85%:** Acceptable — Several gaps, review recommended
- **<80%:** Needs Review — Significant gaps

## Consensus Recommendations

| 🔴 **Critical** | All 3 experts agree | 99% | Implement immediately |
| 🟡 **High** | 2 experts agree | 66% | Review and prioritize |
| 🔵 **Review** | 1 expert only | 33% | May be false positive, investigate |

- Action: Implement immediately

High (2/3 experts):

- Confidence: 66%
- Action: Review and prioritize

Review (1/3 experts):
- "Consider honeypots" (Red Team only)
- Confidence: 33%
- Action: Investigate (may be false positive)
